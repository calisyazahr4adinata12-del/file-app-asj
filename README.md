# File Upload App 

Web aplikasi upload file Caca.

## Features
- Upload file
- Input nama, email, judul file
- Confetti animation
- Sparkle dan shooting star
- Walking cat paw prints

## Tech Stack
- React (Vite)
- Node.js
- PostgreSQL
- MinIO
- Docker
- CRUD

## Cara menjalankan

```bash
docker compose up -d
