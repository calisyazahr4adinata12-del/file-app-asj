# File Upload App

Web aplikasi upload file Caca berbasis CRUD (Create, Read, Update, Delet)

## Features
- Upload file
- Input nama, email, judul file
- Confetti animasi jika file berhasil di upload
- hiasan sparkle di bergound App

## Tech Stack
- React (Vite)
- Node.js
- PostgreSQL
- MinIO
- Docker

## Cara menjalankan

```bash
docker compose up -d
